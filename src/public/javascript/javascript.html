Javascript

Declaring a variable in Javascript

In Javascript, we typically declare variables with either const, for maintaining constant values, or let, for being able to change the value of the variable. Both of these declarations are block scoped, meaning they are only accessible within the block they are declared in.


Printing out to the console

In order to print things to the console, we use console.log().
For example: 
                        
                            const name = "Cristi";
                            console.log( name );

Data types

To begin with, data types in Javascript are dynamic, meaning a variable can be used to hold many different data types.
For example: 
                        
                            let x = 1; // x is a Number
                            let x = "Cristi"; // x is a String

Strings

They are a sequence of characters and can be identified by the use of single or double quotes: 
                        
                            let x = "String";
                            let x = 'Also a string';

Numbers

Unlike other languages, Javascript does not have two different types for int and float/double. Whether or not they have decimals, they are all part of Number. For example: 
                        
                            let x = 1; // without decimals
                            let x = 1.5; // with decimal

Booleans

They can either be true or false.


Arrays

Javascript arrays are indexed from zero; they are written using square brackets [] and their elements are separated by commas ,.
For example: 
                        
                            let array = [ "first", "second", "etc" ];
New elements can be added using array.push( element ). We use array.pop() to remove the last element from the array ( in our example it would be "etc" ).


Copying Array values

One of the ways to copy the values from one array to another, we can use the spread operator ... like so: array2 = [...array1]


Objects

Javascript objects are written using curly braces {} and are structured as key-value pairs, separated by commas.
Example: 
                        
                            let obj = {
                                name: "Cristi",
                                age: 9,
                                isHuman: true
                            }

Undefined and Null

In Javascript, variables without values are set to undefined. undefined and null are the equal in value, but different in type, as null is an object and undefined is of type undefined.


Type checking and conversion

The type of any variable can be checked using the typeof operator, which returns the the type of the given variable in a string format.
Example: 
                        
                            typeof 1 // returns "number"
                            typeof "1" // returns "string"
We can manually convert a variable of one type to another type using the constructor function or by using some parsing functions.
Example: 
                        
                            String( true );
                            String( 1 );
                            Number( "400" );
                            parseInt( "3" );

Comparing variables

Comparing two variables can be done with either == or ===, the first one checking only if the left value is equal to the one on the right, while the second variant checks if both the value and the type is the same on both sides: 
                        
                            "1" == 1 // true
                            "1" === 1 // false
                            1 == true // true
                            1 === true // false

Functions

Functions in Javascript can be declared by using the function keyword. 
                        
                            function myFunction( parameter ) {
                                return "Hello " + parameter;
                            }
                            const greeting = myFunction( "Cristi" );
Functions can also be used as variables, like so: 
                        
                            const myVariableFunction = function() {
                                console.log( "Hello !" );
                            }
                            myVariableFunction();

Arrow Functions

                        
                            const myArrowFunction = ( string ) {
                                console.log( string );
                            }
                            myArrowFunction( "Hello !" );

Callback Functions

Functions can also be passed by reference to other functions in order to be used later: 
                        
                            function sayHiLater ( anyFunctionReference ) {
                                anyFunctionReference( "Hello !" );
                            }
                            sayHiLater( myArrowFunction );
    
                            // We can also pass the parameter to use with the function
                            function interact ( genericInteraction, name ) {
                                console.log( genericInteraction( name ) );
                            }
                            interact( myArrowFunction, "Cristi" );
                            // We can also write our own function directly as a parameter
                            interact( ( string ) => "Greetings, " + string, "Alex" );

Node JS

What is Node JS ?

Node JS is a Javascript runtime for building server-side or desktop applications.


Running files with Node

In order to run Javascript files with node, we simply have to run node <fileName> in the terminal ( Note: we need to make sure we are in the directory that contains our file )
