NPM

What is NPM ?

NPM is a package/module manager for Node.js. All the packages that can be installed through NPM can be found on their website.


Installing packages with NPM

First, we need to make sure there is a package.json file in the folder we want to install a new package. If there isn't any, we can make one and write { } to indicate it is an empty json object.
We can install new packages with NPM by using the command: npm install <package name> or simply npm i <package name>
Example: 
 
                        
                            npm install express
By using the -g option, we can install packages globally: 
                        
                            npm install -g nodemon
By using the -D option, we can install the desired package as a development dependency: 
                        
                            npm install -D cross-env
We are using nodemon for automatically restarting the application every time a file has changed, express for making the API and cross-env for being able to set environment variables in a way that is independent of the developer's OS.

After installing the desired dependencies, the package.json should look like this: 
                        
                            {
                                "dependencies": {
                                  "express": "^4.17.1"
                                },
                                "devDependencies": {
                                  "cross-env": "^7.0.3"
                                }
                            }

Using a package

Using any of the installed packages is as easy as writing require ( "<package name>" ); in your node js file.
Example: const express = require( "express" );


What is package.json

The package.json file contains the names and the versions of all of the project's dependencies that are available through NPM.

After installing a new dependency, it will also be written in the package.json file.

By using the command npm install ( or npm i ) from the folder that contains the package.json, we will tell NPM to install all the dependencies mentioned in the file along with their own dependencies.
By default, they will be installed under ./node_modules/.


Creating scripts with NPM

Within package.json we can also define scripts which can later be used through npm run. Example: 
                        
                            {
                                "scripts": {
                                  "start-dev": "cross-env PORT=3000 nodemon src/app.js"
                                },
                                "dependencies": {
                                  "express": "^4.17.1"
                                },
                                "devDependencies": {
                                  "cross-env": "^7.0.3"
                                }
                            }
We can now run the start-dev script like so: npm run start-dev.
It will create an environment variable with the port number by using cross-env and execute the command nodemon src/app.js to run my application.
